# Image Encryption & Decryption Tool

A Python-based tool for encrypting and decrypting images using XOR operation and pixel shuffling.  
Designed for educational and internship purposes, this project demonstrates *symmetric key encryption* on images.

---

## Features
- Encrypt images using a secret key
- Decrypt images using the same key
- Fully scrambled encryption for strong obfuscation
- User-defined input and output image paths
- Works with PNG, JPG, and JPEG images
- Command-line interface (CLI)

---

## Technologies Used
- Python 3
- Pillow (PIL)
- Random module
